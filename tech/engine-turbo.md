# 涡轮增压发动机 (Turbocharged Engine)

## 什么是涡轮增压

涡轮增压(Turbocharger)是一种利用发动机排出的废气驱动涡轮，压缩进气以提高发动机功率的技术。

## 工作原理

```
排气歧管 → 废气推动涡轮 → 涡轮带动压气机 → 压缩空气进入气缸
                                                    ↓
                                        更多燃油 + 更多空气 = 更大动力
```

### 核心部件
1. **涡轮 (Turbine)** - 被排气推动旋转
2. **压气机 (Compressor)** - 压缩进气
3. **中间体 (Center Housing)** - 连接涡轮和压气机，含轴承
4. **废气旁通阀 (Wastegate)** - 控制增压压力

## 优缺点

### 优点
- 同排量下动力更强（通常提升 30-50%）
- 小排量也能获得大扭矩
- 燃油经济性优于同功率自然吸气发动机
- 高海拔地区动力衰减小

### 缺点
- 涡轮迟滞 (Turbo Lag) - 低转速时响应慢
- 维护成本较高
- 对机油品质要求高
- 涡轮工作温度高，需要良好的散热

## 常见排量与马力参考

| 排量 | 涡轮数量 | 典型马力 | 代表车型 |
|------|---------|---------|---------|
| 1.0T | 单涡轮 | 100-130hp | 福特 EcoBoost |
| 1.5T | 单涡轮 | 150-180hp | 本田思域、大众朗逸 |
| 2.0T | 单涡轮 | 190-250hp | 大众高尔夫GTI、宝马3系 |
| 2.5T | 双涡轮 | 300-400hp | 保时捷 Macan |
| 3.0T | 双涡轮 | 400-550hp | 宝马 M3、奔驰 C63 |
| 4.0T V8 | 双涡轮 | 500-700hp | 保时捷 911 Turbo、迈凯伦 |

## 双涡轮 vs 单涡轮

- **单涡轮 (Single Turbo)** - 结构简单，成本低，常见于家用车
- **双涡轮 (Twin Turbo)** - 两个涡轮串联或并联，减少迟滞，常见于高性能车
- **双涡管 (Twin-Scroll)** - 一个涡轮但两根排气管，兼顾性能和成本

## 涡轮增压 vs 自然吸气 vs 机械增压

| 特性 | 涡轮增压 | 自然吸气 | 机械增压 |
|------|---------|---------|---------|
| 动力提升 | 高 | 低 | 中 |
| 响应速度 | 有迟滞 | 即时 | 即时 |
| 油耗 | 较低 | 较高 | 中等 |
| 维护成本 | 较高 | 较低 | 较高 |
| 代表品牌 | 大众、宝马 | 本田、马自达 | 奔驰(部分) |
