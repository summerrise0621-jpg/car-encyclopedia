# 🚗 汽车百科 (Car Encyclopedia)

汽车知识、品牌历史、技术资料的开放百科全书。

## 📁 目录结构

```
├── brands/      # 品牌介绍（奔驰、宝马、丰田等）
├── tech/        # 汽车技术（发动机、变速箱、新能源等）
├── history/     # 汽车发展历史
├── guides/      # 购车指南、养车知识
└── README.md
```

## 🏷️ 内容分类

### 品牌 (brands/)
- 德系：奔驰、宝马、奥迪、大众
- 日系：丰田、本田、日产、马自达
- 美系：福特、特斯拉、通用
- 国产：比亚迪、吉利、蔚来、小鹏

### 技术 (tech/)
- 发动机技术（涡轮增压、自然吸气、混动）
- 变速箱（AT、CVT、DCT、手动）
- 新能源（纯电动、插电混动、氢燃料）
- 智能驾驶（L1-L5 自动驾驶分级）

### 历史 (history/)
- 汽车诞生与发展
- 各品牌创立故事
- 经典车型回顾

### 指南 (guides/)
- 新车选购建议
- 二手车评估
- 日常保养知识
- 保险与年检
